# biblioteca

## Regras de nomeação

- Arquivos e variáveis snake_case
- Classes PascalCase

## Como rodar (Windows PowerShell)

- Execute a interface gráfica com:

```powershell
python interface\\main.py
```

- Ou via módulos (mais robusto):

```powershell
python -m interface.main
```

Observação: certifique-se de ter Python instalado e disponível no PATH.
